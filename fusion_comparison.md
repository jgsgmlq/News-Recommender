# 多模态融合方法对比

## 方法1: 简单拼接（Concat）

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ID (256维)│  │LLM(256维)│  │GNN(256维)│
└─────┬────┘  └─────┬────┘  └─────┬────┘
      │             │              │
      └─────────────┼──────────────┘
                    ↓
              [拼接: 768维]
                    ↓
            ┌───────────────┐
            │  MLP 降维      │
            │  768 → 256维   │
            └───────────────┘
```

**特点**：
- ✅ 简单直接
- ❌ 固定权重，无法自适应
- ❌ 所有新闻用同样的融合方式

**性能**：AUC = 0.5621


## 方法2: 门控融合（Gate）

```
┌──────────┐  ┌──────────┐  ┌──────────┐
│ID (256维)│  │LLM(256维)│  │GNN(256维)│
└─────┬────┘  └─────┬────┘  └─────┬────┘
      │             │              │
      └─────────────┼──────────────┘
                    ↓
              [拼接: 768维]
                    ↓
         ┌──────────────────┐
         │ 门控网络 (MLP)    │
         │ 768 → 3          │
         │ Softmax          │
         └──────────────────┘
                    ↓
         [w1=0.4, w2=0.35, w3=0.25]
                    ↓
       最终表示 = w1*ID + w2*LLM + w3*GNN
```

**特点**：
- ✅ 可学习权重
- ✅ 每条新闻有不同权重
- ❌ 需要拼接所有模态（计算开销大）

**性能**：AUC = 0.5638


## 方法3: 自适应注意力门控（Attention）⭐

```
           ┌──────────────────────┐
           │ 可学习查询向量 Q      │
           │    (1, 256维)        │
           └──────────┬───────────┘
                      ↓
┌──────────┐  ┌──────────┐  ┌──────────┐
│ID (256维)│  │LLM(256维)│  │GNN(256维)│
└─────┬────┘  └─────┬────┘  └─────┬────┘
      │             │              │
      └─────────────┼──────────────┘
                    ↓
           [堆叠: (3, 256维)]
                    ↓
         ┌──────────────────┐
         │ Key变换 (Linear)  │
         │ (3, 256) → (3, 256)│
         └──────────┬─────────┘
                    ↓
          Q @ K^T  (1,256) × (256,3)
                    ↓
            [注意力得分: (1,3)]
                    ↓
       Softmax / √d  (缩放点积注意力)
                    ↓
         [w1=0.42, w2=0.38, w3=0.20]
                    ↓
       最终表示 = Σ wi * 模态i  (256维)
```

**特点**：
- ✅ 基于注意力机制，更强的表达能力
- ✅ 可学习全局查询向量（学习"哪种模态更重要"）
- ✅ 每条新闻动态权重
- ✅ 缩放点积注意力，训练稳定

**性能**：AUC = 0.5651（最佳）
